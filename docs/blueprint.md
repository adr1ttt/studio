# **App Name**: Kolkata Blaugrana Hub

## Core Features:

- Event Calendar: Display a calendar view of upcoming FC Barcelona fan events in Kolkata.
- Event Booking: Allow users to register for events through a simple form integrated with Razorpay for payments.
- Membership Subscription: Enable users to subscribe as premium members with a payment link integrated with Supabase.
- Member Database: Store member information securely in a Supabase database.
- Contact Form: A simple form for fans to send messages to the admin team.
- Social Media Integration: Display links to Instagram, WhatsApp, X, and Facebook prominently.

## Style Guidelines:

- Primary color: Barcelona blue (#004D98) to evoke the team's identity.
- Background color: Light blue (#D9EAF7), a desaturated shade of the primary color, for a calming backdrop.
- Accent color: A vibrant yellow (#FFDA63), analogous to blue, for interactive elements and call-to-action buttons, ensuring high contrast.
- Headline font: 'Space Grotesk' (sans-serif) for a modern, techy feel.
- Body font: 'Inter' (sans-serif) to complement the headlines and ensure readability.
- Use soccer-themed icons to represent various site functions and Barcelona's legacy.
- Implement a clear, intuitive layout that puts the most important sections front and center.
- Use subtle transitions to add a sense of polish, especially when displaying the event information.
